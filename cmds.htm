Shield Command Guide

General Moderation Commands ▼

ban <user> <r>
Ban a user from the server.
kick <user> <r>
Kick a user from the server.
warn <user> <r>
Warn a user and optionally mute for severe reasons.
unban <id>
Unban a user using their ID or username#discriminator.
mute <user> <time> <r>
Mute a user for a specified time.
unmuteall
Unmute all muted users.

Message Moderation Commands ▼

purge <number>
Purge a specific number of messages.
purgebot/pb
Purge messages from bots.
purgekeyword <keyword> <num>
Purge messages containing a specific keyword.
purgeuser <user> <num>
Purge messages from a specific user.
snipe
Snipe the last deleted message.
editsnipe <num>
Snipe the last edited messages up to n numbers (Admin-only).

User Moderation Commands ▼

chatban <user>
Ban a user from chat and delete every new message from them.
chatunban <user>
Unban a user from chat.
reactban <user>
Ban a user from reacting in channels.
reactunban <user>
Unban a user from reacting in channels.

Server Moderation Commands ▼

lock <chnl>
Lock a specific channel.
unlock <chnl>
Unlock a specific channel.
hide <chnl>
Hide a specific channel (Admin-only).
unhide <chnl>
Unhide a specific channel (Admin-only).
nuke
Reset a channel’s messages completely.
config <override> <user/role/everyone> enable/disable
Change override of a channel.

Voice Channel Moderation Commands ▼

vcban <user>
Ban a user from joining voice channels.
vcunban <user>
Unban a user from joining voice channels.
vcdeafenall
Deafen all users in a specified voice channel.
vckick <user>
Kick a specific user from a voice channel.
vckickall
Kick all users from a specified voice channel.
vcpull <user>
Pull a user to your voice channel.
vcpullall <chnlid>
Move all users from a specified voice channel to your channel.
vcmuteall
Mute all users in a specified voice channel.
vcunmuteall
Unmute all users in a specified voice channel.
vcmove <user> <chnlid>
Move a user to a different voice channel.
vcmoveall <chnlid>
Move all users from your current voice channel to the target voice channel.
vcmute <user>
Mute a specific user in a voice channel.
vcundeafenall
Undeafen all users in a specified voice channel.

Premium Commands ▼

synccategory <categoryid>
Sync permissions across all channels in a category.
targetlock <user>
Locks a specific user’s access.
release <user>
Unlocks all channels for a specific user (Admin-only).
